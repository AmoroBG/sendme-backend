SendMe

Sendme is a platform that connect busy people to people ready and ready to run errands for them.

Endpoints for User

  • POST - Create/ Register user
  • GET - View all users
  • GET - View user
  • DELETE - Delete users
  • DELETE - Delete user
  • PUT - Update a user records
  • PATCH - Update a user record
  • POST - Login
  • GET - Logout

Endpoints for Task

  • POST - Create Task
  • GET - View all Tasks
  • GET - View Task
  • DELETE - Delete Tasks
  • DELETE - Delete Task
  • PUT - Update a Task records
  • PATCH - Update a Task record
